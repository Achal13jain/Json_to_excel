# Changelog

All notable changes to this project will be documented in this file.

## [1.1.0] - 2026-02-14
### Added
- **Try Example**: New "Instant Gratification" feature with 3 sample datasets (E-commerce, API, Analytics).
- **Paste JSON**: Added ability to paste raw JSON text directly in the UI.
- **Viral Sharing**: Added "Share on Twitter" button.
- **Data Fallbacks**: Improved robustness of AI model selection (Gemini 2.5).

### Changed
- **UI Upgrade**: Enhanced `FileUpload` component with tabbed interface.
- **Documentation**: Overhauled README with better visuals and "Trust Signals".

## [1.0.0] - 2026-01-21
### Added
- Initial Release.
- Drag & Drop File Upload.
- Intelligent AI Analysis with Google Gemini.
- Excel Export (XLSX).
- Glassmorphism Design System.
